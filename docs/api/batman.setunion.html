Batman.SetUnion extends Batman.BinarySetOperation. It implements a union operation: the SetUnion contains members found in either set passed to the constructor, without duplicates.

\n

Since Batman.SetUnion is a subclass of Batman.Set, you can use it just like a Batman.Set.

\n
test 'SetUnion contains the union of two sets', ->\n  left = new Batman.Set(1, 2, 3)\n  right = new Batman.Set(3, 4, 5)\n  operation = new Batman.SetUnion(left, right)\n  ok operation.has(1), 'has items from the left set'\n  ok operation.has(4), 'has items from the right set'\n  ok operation.has(3), 'has items from both sets'
