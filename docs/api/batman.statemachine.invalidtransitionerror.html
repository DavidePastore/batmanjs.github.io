Batman.StateMachine.InvalidTransitionError should be thrown when a state machine makes an invalid transition. Batman.StateMachine doesn't throw these errors automatically, but applications of the StateMachine may throw them. For example Batman.Model throws an InvalidTransitionError when save is called on a deleted record.
